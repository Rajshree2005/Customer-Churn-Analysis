# Customer Churn Analysis & Retention Intelligence System

## Project Overview
This project predicts customer churn risk to help the business improve retention. I used a Random Forest classifier to model churn, achieving [X]% accuracy. To bridge the gap between technical insights and business decision-making, I built an interactive application using Streamlit.

## Features
- **Predictive Modeling**: Uses Random Forest to identify high-risk customers.
- **Explainable AI**: Provides factors influencing the churn risk (e.g., high monthly charges, contract type).
- **Interactive Dashboard**: A Streamlit interface for real-time risk assessment.

## Technologies Used
- Python (Pandas, Scikit-learn, Joblib)
- Streamlit
- Random Forest Algorithm

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
